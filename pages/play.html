Play

There are currently no public games (functionality expected in Creative Project 4). However, feel free to try the dedicated demo game ID apitest! For further testing, I recommend the following:
  1. Go to the create page.
  2. Enter any alphanumeric ID into the first field.
  3. Press "Rebuild Project" and wait for around a couple minutes for a JS alert (the API is slow).
  4. Once it is confirmed that the project has been built, return here and enter that same ID to view the demo project.
